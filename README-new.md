## 🚀 Come Iniziare

1. **Clone del repository:**
   ```bash
   git clone https://github.com/tasuboyz/template.git
   cd template
   ```

2. **Apri in browser:**
   ```bash
   # Apri index.html nel tuo browser preferito
   open index.html
   ```

3. **Sviluppo locale:**
   ```bash
   # Usa un server locale per development
   python -m http.server 8000
   # oppure
   npx serve .
   ```
